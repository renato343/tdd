User Story

As a user I want to be able to get a user by Id

  1. Given a valid id a user should be returned
  2. Given an invalid id should return "Id not Valid"
  3. Given an Valid but not existing Id should return "User not Found"

As a user I want to be able to get a list of users

  1. Given a valid request should return a list of users
  2. Given a valid request but no users found should return "No users Found"
